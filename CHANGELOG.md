# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-05

### Added

#### Authentication
- Login page dengan animated gradient wallpaper
- Floating particles effect background
- Session persistence menggunakan localStorage
- User info display di sidebar
- Logout functionality

#### Dashboard
- KPI Cards dengan animasi counting
- Trend Chart untuk visualisasi kunjungan
- Category Chart untuk peminjaman per kategori
- Date Range Picker untuk filter data
- Refresh button untuk update manual
- Loading states dan error handling

#### Visitors Analysis
- Peak Hours Heatmap
- Faculty Distribution Pie Chart
- Visitor Table dengan search dan sort
- Visitor Statistics Cards

#### Loans Analysis
- Top 10 Books Grid dengan ranking
- Category Bar Chart
- Loan Trend Chart
- Late Returns Statistics

#### Recommendation System
- Trending Books (frequency-based)
- Collaborative Filtering recommendations
- Content-Based recommendations
- Algorithm explanation section

#### API Integration
- Service layer untuk Gate System API
- Service layer untuk SLiMS API
- Custom hooks untuk data fetching
- Support mode dummy dan production
- Auto-refresh functionality

#### Common Components
- LoadingSpinner, LoadingCard, LoadingPage
- ErrorMessage dengan variants
- RefreshButton
- DateRangePicker
- DataModeIndicator

### Technical
- React 18 dengan functional components
- Tailwind CSS untuk styling
- Recharts untuk visualisasi
- Lucide React untuk icons
- Environment-based configuration
