# Images Folder

This folder is reserved for uploaded images used in the application.

## Supported Formats
- JPG / JPEG
- PNG
- GIF
- SVG
- WebP

## Usage

Place your images here and reference them in components:

```jsx
<img src="/images/your-image.png" alt="Description" />
```

## Naming Convention

Use kebab-case for image names:
- ✅ `library-logo.png`
- ✅ `book-cover-001.jpg`
- ❌ `Library Logo.png`
- ❌ `bookCover001.jpg`
